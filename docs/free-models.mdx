## Overview

OCA works with any LLM provider. Many offer generous free tiers — enough for daily development, learning, and automation on Android with zero spend.

<Info>
  Free tiers usually allow data collection for model improvement. Avoid sending secrets, private keys, or proprietary code on free plans.
</Info>

## Quick Setup by Provider

### GitHub Copilot (Recommended for beginners)

Free with any GitHub account — no credit card.

```bash
# Install GitHub Copilot CLI
npm install -g @githubnext/github-copilot-cli

# Authenticate
github-copilot-cli auth

# Use model picker
github-copilot-cli /models
```

Free quota: **2,000 code completions/month · 50 chat messages/day**

---

### Google Gemini CLI (Best free quota)

1,500 requests/day free — no card needed.

```bash
npm install -g @google/gemini-cli
gemini  # prompts Google account login on first run
```

Or with API key from [aistudio.google.com](https://aistudio.google.com/apikey):
```bash
export GEMINI_API_KEY=your_key
gemini
```

Free quota: **1,500 req/day · 1M tokens/min** on Gemini 2.0 Flash

---

### Groq (Fastest free inference)

Fastest open-source model inference, free tier is generous.

```bash
# Get free API key at groq.com/keys
export GROQ_API_KEY=your_key

# Use via OpenRouter or direct API in any CLI that supports OpenAI-compatible endpoints
```

Free models: `llama-3.1-8b-instant`, `llama-3.1-70b-versatile`, `gemma2-9b-it`, `qwen-2.5-72b`
Free quota: **14,400 requests/day**

---

### OpenRouter (Most model variety)

Access 50+ free models from one API key.

```bash
# Get free key at openrouter.ai/keys
export OPENROUTER_API_KEY=your_key
```

Look for models ending in `:free` — e.g. `meta-llama/llama-3.1-8b-instruct:free`, `qwen/qwen-2.5-72b-instruct:free`

Free quota: varies per model — most allow 20–200 req/min

---

### OpenCode / Zen (Best for OpenCode CLI users)

```bash
# Register free at zen.opencode.ai — no card needed
# Then in OpenCode CLI:
/models  # pick any model with green tick or 0x label
```

Free models: **MiniMax, Kimi, GLM-5, Big Pickle, Nemotron**
Free quota: **200 req / 5 hours**

## Which Provider Should I Use?

| Goal | Best Provider |
|------|--------------|
| Just starting out | **GitHub Copilot** (free GitHub account, zero config) |
| Maximum daily requests | **Groq** (14,400/day free) or **Gemini CLI** (1,500/day) |
| Best model variety | **OpenRouter** (50+ free models) |
| Using OpenCode/Zen CLI | **OpenCode Zen** |
| Privacy-sensitive work | **Groq** (better privacy than cloud-training providers) |
| Offline fallback | See [Cloud LLM on Android](/docs/local-llm) for Ollama cloud models |

## Usage Tips

- **Check `/models`** in your CLI — green tick or `0x` label = free
- **Rotate providers** when you hit quotas — keep 2–3 API keys ready
- **Gemini CLI** on Termux: set `GEMINI_CLI_TRUST_WORKSPACE=true` for non-interactive use
- **OpenRouter** accepts the same OpenAI SDK format — works with any CLI that supports custom base URL
- Free tiers reset daily or per rolling window — check provider dashboard for exact timing

## Privacy Reminder

<Warning>
  Free tiers on most providers (GitHub, Google, OpenCode) use your prompts for model improvement. Do NOT send private keys, passwords, database credentials, or unreleased proprietary code. Use paid tiers or self-hosted models for sensitive work.
</Warning>
