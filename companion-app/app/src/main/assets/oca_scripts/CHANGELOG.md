# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [010426] - 2026-04-01

### Changed

- **RTX⚡ Protocol Transition**: Rebranded all dependency layers to the new tiered architecture (RTX⚡1: Infrastructure, RTX⚡2: Platform Runtime, RTX⚡3: Master Arsenal).
- Full audit and alignment of all orchestrators (`install.sh`, `update-core.sh`, `uninstall.sh`).
- Enhanced `verify-install.sh` with tiered RTX validation support.
- Deep-rebranded all documentation (README, MDX docs, Technical Guides).

### Removed

- Redundant legacy patch scripts in `patches/` to streamline repository weight.

## [1.0.0] - 2026-04-01

### Added

- Initial release of OpenClaw On Android (OCA)
- Improved cross-platform directory structures (.oca)
- Simplified installation routines
- Seamless Termux integration
